# 跑马灯渐变同步修复说明

## 问题描述

当跑马灯和渐变动画方向一致时（例如都是从右往左），会出现视觉上的参照系问题：
- 文字在向左滚动
- 渐变也在向左移动
- 但因为渐变是相对于文字的，所以看起来渐变像是在向右移动（相对于移动的文字）

这是正常的物理现象，类似于在移动的火车上看窗外的风景。

## 用户尝试的解决方案

用户尝试将 `animationDuration` 从 4.5 秒改为 0.3 秒：
```swift
self.animationDuration = 0.3  // 原来是 4.5
self.marqueeSpeed = 45
```

结果：
- ✅ 渐变方向看起来正确了（因为速度快到看不出相对运动）
- ❌ 但渐变动画变得非常快，有点"鬼畜"

## 最终解决方案（可选的同向速度）

添加 `sameDirectionAnimationDuration` 属性，允许用户为方向一致的情况设置不同的速度：

```swift
/// 当渐变和跑马灯方向一致且跑马灯运行时，使用的动画时长（可选）
/// - 如果设置了此值，当方向一致且跑马灯运行时，会使用此时长替代 animationDuration
/// - 如果为 nil（默认），则始终使用 animationDuration
/// - 建议设置为比 animationDuration 更小的值，以减少视觉冲突
var sameDirectionAnimationDuration: TimeInterval? = nil
```

## 使用方法

### 场景 1：方向一致，设置不同速度（推荐）⭐⭐⭐⭐⭐

```swift
let label = WaveGradientLabelView()
label.text = "这是一段很长的文字需要滚动显示"
label.gradientDirection = .horizontalLeftToRight  // 从左往右
label.marqueeDirection = .rightToLeft             // 从右往左（方向一致）
label.animationDuration = 4.5                     // 正常速度
label.sameDirectionAnimationDuration = 0.5        // ✅ 方向一致时使用 0.5 秒
label.marqueeSpeed = 45
label.enableMarquee = true
label.startAnimation()

// 效果：
// - 跑马灯未启动：渐变速度 = 4.5 秒
// - 跑马灯启动后：渐变速度 = 0.5 秒（自动切换）
// - 跑马灯停止后：渐变速度 = 4.5 秒（自动恢复）
```

### 场景 2：不设置，使用默认速度

```swift
let label = WaveGradientLabelView()
label.text = "这是一段很长的文字需要滚动显示"
label.gradientDirection = .horizontalLeftToRight
label.marqueeDirection = .rightToLeft
label.animationDuration = 4.5
label.marqueeSpeed = 45
// ❌ 不设置 sameDirectionAnimationDuration
label.enableMarquee = true
label.startAnimation()

// 效果：
// - 始终使用 animationDuration = 4.5 秒
// - 会出现视觉冲突（渐变看起来反了）
```

### 场景 3：方向不一致，不需要设置

```swift
let label = WaveGradientLabelView()
label.text = "这是一段很长的文字需要滚动显示"
label.gradientDirection = .horizontalLeftToRight  // 从左往右
label.marqueeDirection = .leftToRight             // 从左往右（方向相反）
label.animationDuration = 4.5
label.sameDirectionAnimationDuration = 0.5        // 设置了也不会用
label.marqueeSpeed = 45
label.enableMarquee = true
label.startAnimation()

// 效果：
// - 始终使用 animationDuration = 4.5 秒
// - 因为方向不一致，不会出现视觉冲突
```

## 实现原理

### 1. 检测方向是否一致

```swift
private func isGradientAndMarqueeDirectionSame() -> Bool {
    switch (gradientDirection, marqueeDirection) {
    case (.horizontalLeftToRight, .rightToLeft):
        return true  // 都向左移动
    case (.horizontalRightToLeft, .leftToRight):
        return true  // 都向右移动
    default:
        return false
    }
}
```

### 2. 获取当前应该使用的时长

```swift
private func getCurrentAnimationDuration() -> TimeInterval {
    let isMarqueeRunning = enableMarquee && marqueeDisplayLink != nil
    
    // 如果跑马灯运行 && 方向一致 && 设置了 sameDirectionAnimationDuration
    if isMarqueeRunning && isGradientAndMarqueeDirectionSame(),
       let sameDirDuration = sameDirectionAnimationDuration {
        return sameDirDuration
    }
    
    // 其他情况使用 animationDuration
    return animationDuration
}
```

### 3. 在动画更新中使用

```swift
@objc private func updateGradientFrame() {
    let duration = max(getCurrentAnimationDuration(), 0.001)
    // ... 使用 duration 计算动画
}
```

## 速度建议

根据实际测试，推荐以下配置：

### 快速滚动（marqueeSpeed = 40-50）

```swift
label.animationDuration = 4.5
label.sameDirectionAnimationDuration = 0.5  // 推荐 0.5 秒
```

### 中速滚动（marqueeSpeed = 20-30）

```swift
label.animationDuration = 3.0
label.sameDirectionAnimationDuration = 0.8  // 推荐 0.8 秒
```

### 慢速滚动（marqueeSpeed = 10-20）

```swift
label.animationDuration = 2.0
label.sameDirectionAnimationDuration = 1.0  // 推荐 1.0 秒
```

## 优点

1. **灵活性**：用户可以自己决定速度，不是强制的
2. **向后兼容**：默认为 nil，不影响现有代码
3. **精确控制**：可以根据实际效果调整速度
4. **自动切换**：跑马灯启动/停止时自动切换速度
5. **只影响水平方向**：垂直和斜向渐变不受影响

## 注意事项

1. **只在方向一致时生效**：
   - `.horizontalLeftToRight` + `.rightToLeft` ✅
   - `.horizontalRightToLeft` + `.leftToRight` ✅
   - 其他组合 ❌

2. **只在跑马灯运行时生效**：
   - 文字不需要滚动时，使用 `animationDuration`
   - 跑马灯停止后，自动恢复 `animationDuration`

3. **可以动态修改**：
   ```swift
   // 运行时修改会立即生效
   label.sameDirectionAnimationDuration = 0.3
   ```

4. **建议值范围**：
   - 太快（< 0.3 秒）：会"鬼畜"
   - 太慢（> 2.0 秒）：效果不明显
   - 推荐：0.5 - 1.0 秒

## 完整示例

```swift
class GMGradientNameView: WaveGradientLabelView {
    func configUI(font: UIFont, level: Int, defaultColors: [UIColor]) {
        self.gradientDirection = .horizontalLeftToRight
        self.font = font
        self.gradientColors = getColors(level: level, defaultColors: defaultColors)
        
        if level > 0 {
            self.animationDuration = 4.5              // 正常速度
            self.sameDirectionAnimationDuration = 0.5 // ✅ 方向一致时使用 0.5 秒
            self.marqueeSpeed = 45
            self.marqueeDelay = 1.0
            self.marqueeDirection = .rightToLeft
            self.enableMarquee = true
        }
    }
}
```
