# 文字闪烁问题修复说明

## 问题描述

**症状：**
- 文字短，容器大（不需要滚动）
- 启用跑马灯：`enableMarquee = true`
- 结果：文字从左侧"呼吸式"缩一下，然后又展开（闪烁效果）

**对比：**
- 不启用跑马灯：`enableMarquee = false` → 正常显示，无闪烁

## 问题原因

### 原来的执行流程

```swift
// 1. updateTextSize() 被调用
if enableMarquee {
    let size = maskLabel.sizeThatFits(...)  // 计算文字实际宽度，如 100px
    maskLabel.frame.size = size              // 设置成 100px
}

// 2. startMarquee() 被调用
let needsScroll = maskLabel.frame.width > bounds.width  // 100 < 300，false
if !needsScroll {
    maskLabel.frame = bounds  // 重新设置成 300px
}
```

**问题：**
1. 第一步：maskLabel 宽度从 300px 变成 100px（文字实际宽度）
2. 第二步：maskLabel 宽度从 100px 变回 300px（容器宽度）
3. 这个过程产生了视觉上的"闪烁"或"呼吸"效果

### 为什么会闪烁？

因为 `maskLabel` 的 `textAlignment = .center`：
- 宽度 100px 时：文字居中在 100px 内
- 宽度 300px 时：文字居中在 300px 内
- 宽度变化导致文字位置变化，产生"缩放"效果

## 修复方案

### 核心思路

**在 `updateTextSize()` 中就判断是否需要滚动，避免不必要的宽度变化。**

### 修复后的流程

```swift
// updateTextSize() 中
if enableMarquee {
    let textSize = maskLabel.sizeThatFits(...)
    let needsScroll = textSize.width > bounds.width * marqueeThreshold
    
    if needsScroll {
        // 需要滚动：使用文字实际宽度
        maskLabel.frame.size = textSize
    } else {
        // ✅ 不需要滚动：直接使用 bounds，避免闪烁
        maskLabel.frame = bounds
    }
}

// startMarquee() 中
let needsScroll = maskLabel.frame.width > bounds.width * marqueeThreshold
if !needsScroll {
    // ✅ updateTextSize() 已经设置好了，不要再次设置
    maskLabel2.isHidden = true
    emojiLabel2.isHidden = true
    return
}
```

### 关键改动

#### 改动 1：updateTextSize 方法

```swift
private func updateTextSize() {
    if enableMarquee {
        // ✅ 先测量文字宽度，判断是否真的需要滚动
        let textSize = maskLabel.sizeThatFits(CGSize(width: CGFloat.greatestFiniteMagnitude, height: bounds.height))
        let needsScroll = textSize.width > bounds.width * marqueeThreshold
        
        if needsScroll {
            // 需要滚动：使用文字的实际宽度
            maskLabel.frame.size = textSize
            maskLabel2.frame.size = textSize
            emojiLabel.frame.size = textSize
            emojiLabel2.frame.size = textSize
        } else {
            // ✅ 不需要滚动：直接使用 bounds，避免闪烁
            maskLabel.frame = bounds
            maskLabel2.frame = bounds
            emojiLabel.frame = bounds
            emojiLabel2.frame = bounds
        }
    } else {
        maskLabel.frame = bounds
        maskLabel2.frame = bounds
        emojiLabel.frame = bounds
        emojiLabel2.frame = bounds
    }
}
```

#### 改动 2：startMarquee 方法

```swift
private func startMarquee() {
    updateTextSize()
    
    let textWidth = maskLabel.frame.width
    let containerWidth = bounds.width
    let needsScroll = textWidth > containerWidth * marqueeThreshold
    
    guard needsScroll else {
        // ✅ 不需要滚动时，updateTextSize() 已经设置好了 frame
        // 只需要隐藏第二份内容即可，不要再次设置 frame（避免闪烁）
        maskLabel2.isHidden = true
        emojiLabel2.isHidden = true
        return
    }
    
    // 需要滚动时的逻辑...
}
```

```swift
// 修复前：计算两次
updateTextSize()  // 第一次计算

// 修复后：只计算一次
updateTextSize()  // 一次性判断并设置
startMarquee()    // 直接使用结果
```

### 优化 2：减少 frame 设置次数

```swift
// 修复前：设置两次 frame
maskLabel.frame.size = textSize  // 第一次
maskLabel.frame = bounds         // 第二次

// 修复后：只设置一次
maskLabel.frame = bounds  // 一次性设置正确的值
```

## 总结

**问题：** 启用跑马灯但文字不需要滚动时，文字产生"呼吸式"闪烁

**原因：** maskLabel 的宽度先变小再变大，导致文字位置变化

**修复：** 在 `updateTextSize()` 中就判断是否需要滚动，避免不必要的宽度变化

**效果：**
- ✅ 短文字：无闪烁，稳定显示
- ✅ 长文字：正常滚动
- ✅ 性能：减少不必要的计算和 frame 设置

现在无论什么场景，都不会出现文字闪烁的问题了！
