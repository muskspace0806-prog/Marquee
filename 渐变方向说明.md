# 渐变方向说明

## 支持的渐变方向

现在支持 8 种渐变方向：

### 1. 水平方向（2种）
```swift
.horizontalLeftToRight  // 左到右
.horizontalRightToLeft  // 右到左 ✨ 新增
```

### 2. 垂直方向（2种）
```swift
.verticalTopToBottom  // 上到下
.verticalBottomToTop  // 下到上 ✨ 新增
```

### 3. 斜向（从上往下，2种）
```swift
.diagonalDownRight  // 左上到右下
.diagonalDownLeft   // 右上到左下
```

### 4. 斜向（从下往上，2种）
```swift
.diagonalUpRight  // 左下到右上
.diagonalUpLeft   // 右下到左上
```

## 方向对比

### 水平方向

```
.horizontalLeftToRight (左→右)
┌─────────────┐
│🔴🟠🟡🟢🔵🟣│  ← 渐变从左边开始，向右移动
└─────────────┘

.horizontalRightToLeft (右→左) ✨
┌─────────────┐
│🟣🔵🟢🟡🟠🔴│  ← 渐变从右边开始，向左移动
└─────────────┘
```

**说明：**
- `horizontalLeftToRight`：渐变色从左侧开始（红色），向右侧过渡（紫色），动画向右移动
- `horizontalRightToLeft`：渐变色从右侧开始（红色），向左侧过渡（紫色），动画向左移动

### 垂直方向

```
.verticalTopToBottom (上→下)
┌───┐
│🔴 │  ← 渐变从顶部开始
│🟠 │
│🟡 │
│🟢 │
│🔵 │
│🟣 │  ← 向下移动
└───┘

.verticalBottomToTop (下→上) ✨
┌───┐
│🟣 │  ← 向上移动
│🔵 │
│🟢 │
│🟡 │
│🟠 │
│🔴 │  ← 渐变从底部开始
└───┘
```

**说明：**
- `verticalTopToBottom`：渐变色从顶部开始（红色），向底部过渡（紫色），动画向下移动
- `verticalBottomToTop`：渐变色从底部开始（红色），向顶部过渡（紫色），动画向上移动

### 示例 5：在 GMGradientNameView 中使用

```swift
class GMGradientNameView: GMWaveGradientLabelView {
    
    func configUI(font: UIFont, level: Int, defaultColors: [UIColor]) {
        stopAnimation()
        enableMarquee = false
        
        // ✅ 使用新的方向名称
        self.gradientDirection = .diagonalDownLeft  // 右上到左下
        
        self.font = font
        self.gradientColors = getColors(level: level, defaultColors: defaultColors)
        
        if level > 0 {
            self.animationDuration = 2.0
            self.marqueeSpeed = 20
            self.marqueeDelay = 1.0
            self.marqueeGap = 10
            
            DispatchQueue.main.async { [weak self] in
                self?.enableMarquee = true
            }
        }
    }
}
```

## 兼容性

### 旧名称仍然可用（已标记为废弃）

```swift
// ⚠️ 旧名称（已废弃，但仍可使用）
.horizontal              // 等同于 .horizontalLeftToRight
.vertical                // 等同于 .verticalTopToBottom
.topLeftToBottomRight    // 等同于 .diagonalDownRight
.topRightToBottomLeft    // 等同于 .diagonalDownLeft

// ✅ 新名称（推荐使用）
.horizontalLeftToRight
.verticalTopToBottom
.diagonalDownRight
.diagonalDownLeft
```

### 迁移指南

```swift
// 旧代码
label.gradientDirection = .horizontal
label.gradientDirection = .vertical
label.gradientDirection = .topLeftToBottomRight
label.gradientDirection = .topRightToBottomLeft

// 新代码（推荐）
label.gradientDirection = .horizontalLeftToRight
label.gradientDirection = .verticalTopToBottom
label.gradientDirection = .diagonalDownRight
label.gradientDirection = .diagonalDownLeft
```

Xcode 会显示警告：
```
'horizontal' is deprecated: renamed to 'horizontalLeftToRight'
'vertical' is deprecated: renamed to 'verticalTopToBottom'
'topLeftToBottomRight' is deprecated: renamed to 'diagonalDownRight'
'topRightToBottomLeft' is deprecated: renamed to 'diagonalDownLeft'
```

## 完整示例

```swift
class ViewController: UIViewController {
    
    override func viewDidLoad() {
        super.viewDidLoad()
        view.backgroundColor = .black
        
        let colors: [UIColor] = [.systemRed, .systemOrange, .systemYellow, 
                                  .systemGreen, .systemBlue, .systemPurple]
        
        // 1. 水平左→右
        let h1 = createLabel(y: 80, text: "水平左→右", direction: .horizontalLeftToRight, colors: colors)
        view.addSubview(h1)
        
        // 2. 水平右→左 ✨
        let h2 = createLabel(y: 140, text: "水平右→左", direction: .horizontalRightToLeft, colors: colors)
        view.addSubview(h2)
        
        // 3. 垂直上→下
        let v1 = createLabel(y: 200, text: "垂直上→下", direction: .verticalTopToBottom, colors: colors)
        view.addSubview(v1)
        
        // 4. 垂直下→上 ✨
        let v2 = createLabel(y: 260, text: "垂直下→上", direction: .verticalBottomToTop, colors: colors)
        view.addSubview(v2)
        
        // 5. 左上→右下
        let dr = createLabel(y: 320, text: "左上→右下", direction: .diagonalDownRight, colors: colors)
        view.addSubview(dr)
        
        // 6. 右上→左下
        let dl = createLabel(y: 380, text: "右上→左下", direction: .diagonalDownLeft, colors: colors)
        view.addSubview(dl)
        
        // 7. 左下→右上
        let ur = createLabel(y: 440, text: "左下→右上", direction: .diagonalUpRight, colors: colors)
        view.addSubview(ur)
        
        // 8. 右下→左上
        let ul = createLabel(y: 500, text: "右下→左上", direction: .diagonalUpLeft, colors: colors)
        view.addSubview(ul)
    }
    
    func createLabel(y: CGFloat, text: String, direction: WaveGradientDirection, colors: [UIColor]) -> WaveGradientLabelView {
        let label = WaveGradientLabelView(frame: CGRect(x: 20, y: y, width: view.bounds.width - 40, height: 50))
        label.text = text
        label.font = UIFont.systemFont(ofSize: 32, weight: .bold)
        label.gradientColors = colors
        label.gradientDirection = direction
        label.animationDuration = 3.0
        label.startAnimation()
        return label
    }
}
```

## 方向选择建议

### 根据设计需求选择

1. **简单场景**：使用 `.horizontalLeftToRight` 或 `.verticalTopToBottom`
2. **反向效果**：使用 `.horizontalRightToLeft` 或 `.verticalBottomToTop` ✨
3. **动感效果**：使用斜向渐变
4. **从上往下**：使用 `.diagonalDownRight` 或 `.diagonalDownLeft`
5. **从下往上**：使用 `.diagonalUpRight` 或 `.diagonalUpLeft`

### 视觉效果对比

```swift
// 水平方向
.horizontalLeftToRight  // 左→右（常规）
.horizontalRightToLeft  // 右→左（反向）✨

// 垂直方向
.verticalTopToBottom    // 上→下（常规）
.verticalBottomToTop    // 下→上（反向）✨

// 从上往下：给人"下沉"的感觉
.diagonalDownRight      // 左上→右下
.diagonalDownLeft       // 右上→左下

// 从下往上：给人"上升"的感觉
.diagonalUpRight        // 左下→右上
.diagonalUpLeft         // 右下→左上
```

## 总结

**新增方向：**
- ✨ `.horizontalRightToLeft` - 水平右到左
- ✨ `.verticalBottomToTop` - 垂直下到上
- ✨ `.diagonalUpRight` - 左下到右上
- ✨ `.diagonalUpLeft` - 右下到左上

**重命名：**
- ~~`.horizontal`~~ → `.horizontalLeftToRight`
- ~~`.vertical`~~ → `.verticalTopToBottom`
- ~~`.topLeftToBottomRight`~~ → `.diagonalDownRight`
- ~~`.topRightToBottomLeft`~~ → `.diagonalDownLeft`

**兼容性：**
- 旧名称仍可使用，但会显示废弃警告
- 推荐使用新名称

**总共支持：**
- 2 个水平方向（左→右、右→左）
- 2 个垂直方向（上→下、下→上）
- 4 个斜向方向（2 个从上往下，2 个从下往上）

现在你有 8 种渐变方向可以选择了！
