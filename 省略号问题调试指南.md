# 省略号问题调试指南

## 问题描述

文字 `(★≧▽^))😃😂201` 被截断后，只显示 `(★≧▽^))`，没有省略号，表情符号也不见了。

期望：`(★≧▽^))😃...`

## 可能的问题和解决方案

### 问题 1：maskLabel 没有显示省略号

**原因**：`lineBreakMode` 没有正确设置为 `.byTruncatingTail`

**解决方案**：
```swift
// 在 updateLineBreakMode() 中添加调试
print("🔧 [updateLineBreakMode] enableMarquee=\(enableMarquee)")
print("   - maskLabel.lineBreakMode: \(maskLabel.lineBreakMode.rawValue)")
```

### 问题 2：emojiLabel 覆盖了省略号

**原因**：`emojiLabel` 虽然设置了 `.byClipping`，但还是显示了黑色省略号

**解决方案**：
```swift
// 方案 A：确保 emojiLabel.textColor 是透明的
emojiLabel.textColor = .clear

// 方案 B：在 attributedText 中为所有字符设置透明颜色
emojiAttr.append(NSAttributedString(string: str, attributes: [
    .font: font,
    .foregroundColor: UIColor.clear  // 明确设置透明
]))
```

### 问题 3：表情符号位置是空白

**原因**：`maskLabel` 中表情符号是透明的，`emojiLabel` 被隐藏或透明

**解决方案**：
```swift
// 方案 A：让 maskLabel 中的表情符号也显示为白色
if isEmoji {
    maskAttr.append(NSAttributedString(string: str, attributes: [
        .font: emojiFont,
        .foregroundColor: UIColor.white  // 改为白色，应用渐变
    ]))
}

// 方案 B：不隐藏 emojiLabel，确保表情符号正常显示
emojiLabel.alpha = 1.0
```

### 问题 4：clipsToBounds 导致省略号被裁剪

**原因**：`clipsToBounds = true` 可能裁剪了省略号

**解决方案**：
```swift
// 临时测试：禁用 clipsToBounds
self.clipsToBounds = false
```

## 最终解决方案（如果上述都不行）

如果所有方案都不行，使用自定义省略号：

```swift
private func addCustomEllipsis(to text: String, maxWidth: CGFloat) -> NSAttributedString {
    let ellipsis = "..."
    let ellipsisWidth = (ellipsis as NSString).size(withAttributes: [.font: font]).width
    
    var result = ""
    var currentWidth: CGFloat = 0
    
    for ch in text {
        let charStr = String(ch)
        let charWidth = (charStr as NSString).size(withAttributes: [.font: font]).width
        
        if currentWidth + charWidth + ellipsisWidth > maxWidth {
            break
        }
        
        result.append(ch)
        currentWidth += charWidth
    }
    
    result += ellipsis
    
    // 构建 attributedText
    let attr = NSMutableAttributedString()
    for ch in result {
        let str = String(ch)
        let isEmoji = characterContainsEmoji(ch)
        
        if isEmoji {
            attr.append(NSAttributedString(string: str, attributes: [
                .font: UIFont.systemFont(ofSize: font.pointSize),
                .foregroundColor: UIColor.white  // 表情符号也应用渐变
            ]))
        } else {
            attr.append(NSAttributedString(string: str, attributes: [
                .font: font,
                .foregroundColor: UIColor.white
            ]))
        }
    }
    
    return attr
}
```
