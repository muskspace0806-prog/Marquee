# 方框字母和装饰字符处理

## 问题描述

用户反馈：`🅼🅰🅽🅾😼😼` 中的前四个方框字母显示为黑色。

## 字符分析

### 方框字母（Enclosed Alphanumerics）

这些是装饰性的字母，通常用于强调或装饰：

| 字符 | Unicode | 名称 |
|------|---------|------|
| 🅰 | U+1F170 | 负方形拉丁大写字母 A |
| 🅱 | U+1F171 | 负方形拉丁大写字母 B |
| 🅾 | U+1F17E | 负方形拉丁大写字母 O |
| 🅿 | U+1F17F | 负方形拉丁大写字母 P |
| 🅼 | U+1F17C | 负方形拉丁大写字母 M |
| 🅽 | U+1F17D | 负方形拉丁大写字母 N |

### 范围

- **方框字母**：`0x1F170...0x1F189`
- **带圆圈数字**：`0x1F100...0x1F10C`

## 问题原因

这些字符在 `0x1F000...0x1FFFF` 范围内，被识别为表情符号。但它们实际上是装饰性的字母，不是真正的彩色表情，所以：
- 被识别为表情后，使用系统字体渲染
- 但系统无法将它们渲染为彩色（因为它们本质上是单色字符）
- 结果显示为黑色

## 解决方案

将这些装饰性字符当作普通文字处理，应用渐变效果。

### 实现代码

```swift
private func isEmojiScalar(_ scalar: UnicodeScalar) -> Bool {
    if scalar.properties.isEmojiPresentation { return true }
    
    if scalar.value >= 0x1F000 && scalar.value <= 0x1FFFF {
        // 排除单独的区域指示符号
        if scalar.value >= 0x1F1E6 && scalar.value <= 0x1F1FF {
            return false
        }
        
        // 排除带圆圈/方框的字母和数字
        if scalar.value >= 0x1F170 && scalar.value <= 0x1F189 {
            return false  // 🅰-🆉 等方框字母
        }
        if scalar.value >= 0x1F100 && scalar.value <= 0x1F10C {
            return false  // 🄀-🄌 等带圆圈数字
        }
        
        return true
    }
    
    return false
}
```

## 渲染效果

### 示例：`🅼🅰🅽🅾😼😼`

**字符分析**：
- `🅼` (U+1F17C) - 方框字母 M → 文字（应用渐变）
- `🅰` (U+1F170) - 方框字母 A → 文字（应用渐变）
- `🅽` (U+1F17D) - 方框字母 N → 文字（应用渐变）
- `🅾` (U+1F17E) - 方框字母 O → 文字（应用渐变）
- `😼` (U+1F63C) - 猫脸表情 → 表情（显示原色）
- `😼` (U+1F63C) - 猫脸表情 → 表情（显示原色）

**修复前**：
- 🅼🅰🅽🅾 显示为黑色

**修复后**：
- 🅼🅰🅽🅾 应用渐变效果
- 😼😼 显示原色（彩色）

### 3. 其他装饰字符

还有一些类似的装饰字符：
- ⓐ-ⓩ (U+24D0-24E9) - 带圆圈小写字母
- Ⓐ-Ⓩ (U+24B6-24CF) - 带圆圈大写字母
- ①-⑳ (U+2460-2473) - 带圆圈数字

这些字符不在 `0x1F000...0x1FFFF` 范围内，已经被正确处理为文字。

## 排除的字符范围总结

为了确保正确的渲染效果，我们排除了以下范围：

| 范围 | 说明 | 处理方式 |
|------|------|----------|
| 0x1F1E6-0x1F1FF | 区域指示符号（单个） | 作为文字 |
| 0x1F170-0x1F189 | 方框字母 | 作为文字 |
| 0x1F100-0x1F10C | 带圆圈数字 | 作为文字 |

这些字符虽然在表情符号范围内，但它们是装饰性字符，应该跟文字一起渲染。

## 注意事项

1. **装饰性字符**：这些字符用于装饰和强调，不是真正的表情
2. **单色渲染**：这些字符本质上是单色的，无法显示为彩色
3. **渐变效果**：将它们当作文字处理，应用渐变效果更合适
4. **视觉统一**：与周围文字一起渲染，视觉效果更统一
