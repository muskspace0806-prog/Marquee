# 跑马灯方向说明

## 功能概述

`WaveGradientLabelView` 支持两种跑马灯滚动方向：
- **从右往左**（默认）：适用于 LTR（Left-to-Right）语言，如中文、英文
- **从左往右**：适用于 RTL（Right-to-Left）语言，如阿拉伯语、希伯来语

## 使用方法

### 1. 从右往左（默认）

```swift
let label = WaveGradientLabelView()
label.text = "这是一段很长的文字，会自动滚动显示"
label.enableMarquee = true
label.marqueeDirection = .rightToLeft  // 从右往左（默认）
label.marqueeSpeed = 20
label.marqueeDelay = 1.0
label.startAnimation()
```

**效果：**
```
┌─────────────────────────┐
│                    文字→│  文字从右边进入
│              文字内容→  │  向左移动
│        文字内容完整→    │
│  文字内容完整显示→      │
│→                        │  文字从左边离开
└─────────────────────────┘
```

### 2. 从左往右（RTL 语言）

```swift
let label = WaveGradientLabelView()
label.text = "مرحبا بك في التطبيق"  // 阿拉伯语
label.enableMarquee = true
label.marqueeDirection = .leftToRight  // 从左往右
label.marqueeSpeed = 20
label.marqueeDelay = 1.0
label.startAnimation()
```

**效果：**
```
┌─────────────────────────┐
│←文字                    │  文字从左边进入
│  ←文字内容              │  向右移动
│    ←文字内容完整        │
│      ←文字内容完整显示  │
│                        ←│  文字从右边离开
└─────────────────────────┘
```

## 属性说明

### marqueeDirection

```swift
var marqueeDirection: MarqueeDirection = .rightToLeft
```

**类型：** `MarqueeDirection` 枚举

**可选值：**
- `.rightToLeft` - 从右往左（默认）
- `.leftToRight` - 从左往右

**默认值：** `.rightToLeft`

**说明：**
- 改变方向时，如果跑马灯正在运行，会自动重新启动
- 适用于多语言应用，根据语言方向自动调整

## 完整示例

### 示例 1：中文/英文（LTR）

```swift
let label = WaveGradientLabelView(frame: CGRect(x: 20, y: 100, width: 300, height: 60))
label.text = "这是一段很长的文字，会自动滚动显示跑马灯效果😀"
label.font = UIFont.systemFont(ofSize: 30, weight: .bold)
label.gradientColors = [.systemPink, .systemPurple, .systemBlue]
label.gradientDirection = .horizontalLeftToRight
label.animationDuration = 2.0

// 跑马灯配置
label.enableMarquee = true
label.marqueeDirection = .rightToLeft  // 从右往左
label.marqueeSpeed = 20                // 滚动速度（像素/秒）
label.marqueeDelay = 1.0               // 延迟时间（秒）
label.marqueeGap = 40                  // 两段文字之间的间距

label.startAnimation()
view.addSubview(label)
```

let label = WaveGradientLabelView(frame: CGRect(x: 20, y: 180, width: 300, height: 60))

### 示例 3：根据语言自动设置方向

```swift
func createMarqueeLabel(text: String, isRTL: Bool) -> WaveGradientLabelView {
    let label = WaveGradientLabelView(frame: CGRect(x: 20, y: 100, width: 300, height: 60))
    label.text = text
    label.font = UIFont.systemFont(ofSize: 30, weight: .bold)
    label.gradientColors = [.systemPink, .systemPurple, .systemBlue]
    
    // 根据语言方向设置跑马灯方向
    label.enableMarquee = true
    label.marqueeDirection = isRTL ? .leftToRight : .rightToLeft
    label.marqueeSpeed = 20
    label.marqueeDelay = 1.0
    
    label.startAnimation()
    return label
}

// 使用
let chineseLabel = createMarqueeLabel(text: "中文文本", isRTL: false)
let arabicLabel = createMarqueeLabel(text: "النص العربي", isRTL: true)
```

### 示例 4：动态切换方向

```swift
let label = WaveGradientLabelView()
label.text = "可以动态切换方向"
label.enableMarquee = true
label.marqueeDirection = .rightToLeft
label.startAnimation()

// 3 秒后切换方向
DispatchQueue.main.asyncAfter(deadline: .now() + 3) {
    label.marqueeDirection = .leftToRight  // 自动重新启动
}
```

## 实现原理

### 从右往左（.rightToLeft）

1. **初始位置：** 文字从容器右边外开始（`x = containerWidth`）
2. **移动方向：** 向左移动（`x -= offset`）
3. **循环逻辑：** 当第一份文字完全离开左边时，第二份文字正好进入右边

```swift
// 初始位置
maskLabel.frame.origin.x = containerWidth
maskLabel2.frame.origin.x = containerWidth + w + gap

// 动画更新
x1 = containerWidth - offset
x2 = x1 + cycleLen
```

### 从左往右（.leftToRight）

1. **初始位置：** 文字从容器左边外开始（`x = -w`）
2. **移动方向：** 向右移动（`x += offset`）
3. **循环逻辑：** 当第一份文字完全离开右边时，第二份文字正好进入左边

```swift
// 初始位置
maskLabel.frame.origin.x = -w
maskLabel2.frame.origin.x = -w - w - gap

// 动画更新
x1 = -w + offset
x2 = x1 - cycleLen
```

## 与其他属性的配合

### 1. 与文字对齐方式配合

```swift
label.marqueeDirection = .leftToRight
label.textAlignment = .right  // RTL 语言通常右对齐
```

### 2. 与渐变方向配合

```swift
// RTL 语言：跑马灯从左往右，渐变也从右往左
label.marqueeDirection = .leftToRight
label.gradientDirection = .horizontalRightToLeft
```

### 3. 与跑马灯速度配合

```swift
// 不同方向可以使用不同的速度
if isRTL {
    label.marqueeDirection = .leftToRight
    label.marqueeSpeed = 25  // RTL 可能需要稍快的速度
} else {
    label.marqueeDirection = .rightToLeft
    label.marqueeSpeed = 20
}
```

## 常见问题

### Q1: 如何判断当前语言是否为 RTL？

```swift
func isRTLLanguage() -> Bool {
    return UIApplication.shared.userInterfaceLayoutDirection == .rightToLeft
}

// 使用
label.marqueeDirection = isRTLLanguage() ? .leftToRight : .rightToLeft
```

### Q2: 切换方向时会闪烁吗？

不会。切换方向时会自动停止并重新启动跑马灯，过程是平滑的。

### Q3: 可以在运行时动态切换方向吗？

可以。直接设置 `marqueeDirection` 属性即可，如果跑马灯正在运行，会自动重新启动。

```swift
label.marqueeDirection = .leftToRight  // 自动重新启动
```

### Q4: 两个方向的速度一样吗？

是的。`marqueeSpeed` 属性对两个方向都有效，单位都是像素/秒。

## 最佳实践

### 1. 根据系统语言自动设置

```swift
extension WaveGradientLabelView {
    func configureForCurrentLanguage() {
        let isRTL = UIApplication.shared.userInterfaceLayoutDirection == .rightToLeft
        self.marqueeDirection = isRTL ? .leftToRight : .rightToLeft
        self.textAlignment = isRTL ? .right : .left
    }
}

// 使用
label.configureForCurrentLanguage()
```

### 2. 多语言应用中的使用

```swift
class LocalizedMarqueeLabel: WaveGradientLabelView {
    
    var localizedText: String = "" {
        didSet {
            self.text = NSLocalizedString(localizedText, comment: "")
            updateDirectionForCurrentLanguage()
        }
    }
    
    private func updateDirectionForCurrentLanguage() {
        let isRTL = UIApplication.shared.userInterfaceLayoutDirection == .rightToLeft
        self.marqueeDirection = isRTL ? .leftToRight : .rightToLeft
    }
}
```

### 3. 性能优化

```swift
// 如果确定不需要切换方向，可以在初始化时设置一次
label.marqueeDirection = .rightToLeft
label.enableMarquee = true  // 最后启用，避免多次重启
```
