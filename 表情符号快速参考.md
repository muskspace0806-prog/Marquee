# 表情符号快速参考

## 渲染规则

### 显示原色（不应用渐变）
- 👑 😀 😯 🐔 🎨 😼 等标准表情
- 🇺🇸 🇨🇳 🇯🇵 等完整国旗（两个区域指示符号）
- 判断：`isEmojiPresentation == true` 或在表情范围且不在排除列表

### 应用渐变（跟文字一起）
- ★ ❤︎ ✓ ♥ 等文本符号
- 🇦 🇧 🇨 等单独的区域指示符号（不完整的国旗）
- 🅰 🅱 🅾 🅿 等方框字母（装饰字符）
- 𓄂 等特殊字符
- 判断：包含 `U+FE0E` 或在排除范围或不在表情范围

## 排除范围（作为文字处理）

| 范围 | 字符示例 | 说明 |
|------|----------|------|
| 0x1F1E6-0x1F1FF | 🇦 🇧 🇨 | 单独的区域指示符号 |
| 0x1F170-0x1F189 | 🅰 🅱 🅾 🅿 | 方框字母 |
| 0x1F100-0x1F10C | 🄀 🄁 🄂 | 带圆圈数字 |

## 示例

```swift
// ★ 和 ❤︎ 应用渐变
label.text = "★𝐓𝐎𝐅𝐀𝐍★ 𝐉𝐚𝐍❤︎"

// 👑 显示原色，文字应用渐变
label.text = "👑𝕄𝕒𝕕𝕒𝕞𝕖"

// 🇺🇸 显示原色，🇦 应用渐变
label.text = "𓄂🇺🇸دیوانه🇦"

// 🅼🅰🅽🅾 应用渐变，😼 显示原色
label.text = "🅼🅰🅽🅾😼😼"

// 调试
label.debugEmojiDetection()
```

## 常见字符分类

| 字符 | 渲染方式 | 说明 |
|------|----------|------|
| 👑 😀 😼 🐔 | 原色 | 标准表情 |
| 🇺🇸 🇨🇳 🇯🇵 | 原色 | 完整国旗 |
| ★ ❤︎ ✓ | 渐变 | 文本符号 |
| 🇦 🇧 🇨 | 渐变 | 不完整国旗 |
| 🅰 🅱 🅾 🅿 | 渐变 | 方框字母 |
| 𝐉𝐚𝐍 𓄂 | 渐变 | 特殊字符 |

## 说明

- **完整国旗**（如 🇺🇸）= 两个区域指示符号 → 显示原色
- **单独字母**（如 🇦）= 一个区域指示符号 → 应用渐变
- **方框字母**（如 🅰）= 装饰字符 → 应用渐变
- **标准表情**（如 😼）= 真正的表情 → 显示原色
