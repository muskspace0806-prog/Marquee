# 重复文字问题修复说明

## 问题描述

**症状：**
- 设置了 `enableMarquee = true`
- 文字：`123456`
- 容器宽度很大，文字不需要滚动
- 结果：显示成 `123456 123`（重复显示）

## 问题原因

跑马灯功能使用了两份内容（`maskLabel` + `maskLabel2`，`emojiLabel` + `emojiLabel2`）来实现无缝循环。

当文字不需要滚动时：
- `startMarquee()` 判断不需要滚动，直接 return
- 但 `maskLabel2` 和 `emojiLabel2` 没有被隐藏
- 结果：两份内容都显示出来了

## 修复方案

### 修复点 1：startMarquee 方法

```swift
private func startMarquee() {
    updateTextSize()
    
    let textWidth = maskLabel.frame.width
    let containerWidth = bounds.width
    let needsScroll = textWidth > containerWidth * marqueeThreshold
    
    guard needsScroll else {
        // ✅ 不需要滚动时，确保第二份内容隐藏
        maskLabel.frame = bounds
        emojiLabel.frame = bounds
        
        // ✅ 关键：隐藏第二份内容
        maskLabel2.isHidden = true
        emojiLabel2.isHidden = true
        
        return
    }
    
    // ✅ 需要滚动时，显示第二份内容
    maskLabel2.isHidden = false
    emojiLabel2.isHidden = false
    
    // ... 启动跑马灯 ...
}
```

### 修复点 2：stopMarquee 方法

```swift
private func stopMarquee() {
    marqueeDisplayLink?.invalidate()
    marqueeDisplayLink = nil
    
    maskLabel.frame = bounds
    maskLabel2.frame = bounds
    emojiLabel.frame = bounds
    emojiLabel2.frame = bounds
    
    // ✅ 兜底：确保第二份内容隐藏
    maskLabel2.isHidden = true
    emojiLabel2.isHidden = true
}
```

### 修复点 3：layoutSubviews 方法

```swift
override func layoutSubviews() {
    super.layoutSubviews()
    
    // ... 其他代码 ...
    
    if !enableMarquee {
        emojiLabel.frame = bounds
        emojiLabel2.isHidden = true
        maskLabel2.isHidden = true  // ✅ 兜底：确保隐藏
    } else {
        emojiLabel.isHidden = false
        emojiLabel2.isHidden = false
        maskLabel2.isHidden = false  // ✅ 确保显示
        
        // ... 其他代码 ...
    }
}
```

## 使用示例

### 正常使用（无需修改）

```swift
let label = WaveGradientLabelView(frame: CGRect(x: 20, y: 100, width: 300, height: 60))
label.text = "123456"  // 短文字
label.enableMarquee = true

// ✅ 自动判断：文字短，不滚动，只显示一次
```

```swift
let label = WaveGradientLabelView(frame: CGRect(x: 20, y: 100, width: 200, height: 60))
label.text = "这是一段很长的文字需要滚动"  // 长文字
label.enableMarquee = true

// ✅ 自动判断：文字长，启动滚动
```

### 在 GMGradientNameView 中使用

```swift
class GMGradientNameView: GMWaveGradientLabelView {
    
    func configUI(font: UIFont, level: Int, defaultColors: [UIColor]) {
        stopAnimation()
        enableMarquee = false
        
        self.gradientDirection = .topRightToBottomLeft
        self.font = font
        self.gradientColors = getColors(level: level, defaultColors: defaultColors)
        
        if level > 0 {
            self.animationDuration = 2.0
            self.marqueeSpeed = 20
            self.marqueeDelay = 1.0
            self.marqueeGap = 10
            
            DispatchQueue.main.async { [weak self] in
                self?.enableMarquee = true
                // ✅ 自动判断：文字短不滚动，文字长才滚动
            }
        }
    }
}
```

## 兜底机制

修复后的代码包含多重兜底：

1. **startMarquee 兜底**：判断不需要滚动时，隐藏第二份内容
2. **stopMarquee 兜底**：停止时，隐藏第二份内容
3. **layoutSubviews 兜底**：布局时，根据 enableMarquee 状态控制显示/隐藏

## 总结

**问题：** 启用跑马灯但文字不需要滚动时，显示重复文字

**原因：** 第二份内容（maskLabel2、emojiLabel2）没有被隐藏

**修复：** 在不需要滚动时，显式隐藏第二份内容

**效果：** 
- ✅ 文字短：只显示一次
- ✅ 文字长：正常滚动
- ✅ 动态切换：自动适应

现在无论什么场景，都不会出现重复文字的问题了！
