# 国旗表情和区域指示符号处理

## 问题描述

用户反馈：`𓄂🇺🇸دیوانه🇦` 中的 `🇦` 显示为方框。

## 原因分析

### 国旗表情的组成

国旗表情由两个区域指示符号（Regional Indicator Symbols）组成：

- `🇺🇸` = `🇺` (U+1F1FA) + `🇸` (U+1F1F8)
- `🇨🇳` = `🇨` (U+1F1E8) + `🇳` (U+1F1F3)
- `🇯🇵` = `🇯` (U+1F1EF) + `🇵` (U+1F1F5)

### 不完整的区域指示符号

`🇦` 只是一个单独的区域指示符号 (U+1F1E6)，没有配对的第二个字母，因此：
- 无法形成完整的国旗
- 系统显示为方框（因为无法渲染）
- 如果被识别为表情，会显示为黑色方框

## 解决方案

### 检测逻辑

1. **完整的国旗**（两个区域指示符号）
   - 作为表情处理
   - 显示原色（彩色国旗）

2. **不完整的区域指示符号**（单个）
   - 作为文字处理
   - 应用渐变效果（避免黑色方框）

### 实现代码

```swift
private func isEmojiScalar(_ scalar: UnicodeScalar) -> Bool {
    if scalar.properties.isEmojiPresentation { return true }
    
    if scalar.value >= 0x1F000 && scalar.value <= 0x1FFFF {
        // 排除单独的区域指示符号
        if scalar.value >= 0x1F1E6 && scalar.value <= 0x1F1FF {
            return false
        }
        return true
    }
    
    return false
}

private func characterContainsEmoji(_ ch: Character) -> Bool {
    if ch.unicodeScalars.contains(where: { $0.value == 0xFE0E }) {
        return false
    }
    
    // 检查是否是完整的国旗
    let regionalIndicators = ch.unicodeScalars.filter { 
        $0.value >= 0x1F1E6 && $0.value <= 0x1F1FF 
    }
    if regionalIndicators.count == 2 {
        return true  // 完整的国旗
    } else if regionalIndicators.count == 1 {
        return false  // 不完整，作为文字处理
    }
    
    for scalar in ch.unicodeScalars {
        if scalar.value == 0xFE0E || scalar.value == 0xFE0F {
            continue
        }
        if isEmojiScalar(scalar) { return true }
    }
    return false
}
```

## 渲染效果

### 示例：`𓄂🇺🇸دیوانه🇦`

**字符分析**：
- `𓄂` (U+13102) - 埃及象形文字 → 文字（应用渐变）
- `🇺🇸` (U+1F1FA + U+1F1F8) - 美国国旗 → 表情（显示原色）
- `دیوانه` - 波斯语/阿拉伯语 → 文字（应用渐变）
- `🇦` (U+1F1E6) - 单独的区域指示符号 → 文字（应用渐变）

**修复前**：
- `🇦` 显示为黑色方框

**修复后**：
- `🇦` 应用渐变效果（不再是黑色方框）

## 区域指示符号范围

区域指示符号在 `0x1F1E6...0x1F1FF` 范围内，对应字母 A-Z：

| 字符 | Unicode | 字母 |
|------|---------|------|
| 🇦 | U+1F1E6 | A |
| 🇧 | U+1F1E7 | B |
| 🇨 | U+1F1E8 | C |
| ... | ... | ... |
| 🇿 | U+1F1FF | Z |

## 常见国旗

| 国旗 | 组成 | Unicode |

| 🇺🇸 | US | U+1F1FA + U+1F1F8 |
| 🇨🇳 | CN | U+1F1E8 + U+1F1F3 |
| 🇯🇵 | JP | U+1F1EF + U+1F1F5 |
| 🇬🇧 | GB | U+1F1EC + U+1F1E7 |

## 其他特殊字符

### 埃及象形文字

- 范围：`0x13000...0x1342F`
- 示例：`𓄂` (U+13102)
- 处理：作为文字，应用渐变

### 组合字符

某些表情由多个 Unicode 字符组成：
- 肤色修饰符：`👋🏻` = `👋` + `🏻`
- ZWJ 序列：`👨‍👩‍👧` = `👨` + ZWJ + `👩` + ZWJ + `👧`

这些组合字符会被正确识别为表情（因为包含 `isEmojiPresentation` 的字符）。

## 注意事项

1. **国旗需要配对**：单独的区域指示符号无法形成国旗
2. **方框问题**：不完整的区域指示符号会显示为方框
3. **渐变处理**：将不完整的区域指示符号当作文字处理，应用渐变
4. **完整国旗**：两个区域指示符号组成的国旗显示原色

通过检测区域指示符号的配对情况：
- 完整的国旗（两个区域指示符号）显示原色
- 不完整的区域指示符号应用渐变，避免黑色方框
- 其他特殊字符（如埃及象形文字）正常应用渐变
